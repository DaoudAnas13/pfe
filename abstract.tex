% \chapter*{Abstract}

% This Master's Final Year Engineering Project (PFE) presents the comprehensive design, development, and implementation of RM - Report Manager, an integrated web and mobile application platform for managing IT equipment installation interventions. The system addresses the critical need for streamlined field intervention management within the telecommunications and IT infrastructure sectors, where traditional paper-based and fragmented digital processes have led to inefficiencies, data inconsistencies, and delayed reporting.

% The project encompasses a full-stack solution comprising a Spring Boot microservices backend powered by JHipster, an Angular-based web application, and a Flutter mobile application, all orchestrated through Docker containers and deployed on scalable cloud infrastructure. The technical architecture leverages PostgreSQL with JSONB data structures for flexible form management, JWT-based authentication for secure multi-tenant access, and Firebase for real-time notifications and offline synchronization.

% Employing SCRUM methodology across five development sprints, the project successfully delivered core functionalities including dynamic form generation, automated Excel report creation, role-based access control, and comprehensive intervention lifecycle management. The implementation demonstrates industry-standard practices in software engineering, including test-driven development, continuous integration, and agile project management.

% The resulting platform serves four primary user roles: field technicians for data collection, supervisors for quality control, coordinators for template management, and administrators for system configuration. Through extensive stakeholder analysis and iterative prototyping, the solution achieves significant improvements in operational efficiency, data accuracy, and user experience compared to existing systems.

% This report documents the complete engineering process from requirements analysis through deployment, providing detailed technical specifications, architectural decisions, implementation details, and validation results. The project not only delivers a production-ready application but also serves as a comprehensive case study in modern full-stack development and agile software engineering practices.
