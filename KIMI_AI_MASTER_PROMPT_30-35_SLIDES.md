  );

**Example 2: Approval Workflow (Spring Boot Service)**
```java
@Service
public class RapportRequestService {
    @Transactional
    public void approveIntervention(Long requestId, String approverEmail) {
        RapportRequest request = repository.findById(requestId).orElseThrow();
        StatusHistory history = StatusHistory.builder()
            .previousStatus(request.getStatus())
            .newStatus(Status.APPROVED)
            .user(userRepository.findByEmail(approverEmail))
            .timestamp(now())
            .build();
        request.setStatus(Status.APPROVED);
        repository.save(request);
        statusHistoryRepository.save(history);
        firebaseService.sendNotification(request.getAssignee(), "Your intervention was approved!");
    }
}
```
